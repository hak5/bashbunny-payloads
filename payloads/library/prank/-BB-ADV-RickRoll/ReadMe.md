# ADV-RickRoll

A script used to do an advanced rick roll on your target

## Description

This program rick rolls your target without opening a muted youtube video

## Getting Started

### Dependencies

* An internet connection
* Windows 10,11

<p align="right">(<a href="#top">back to top</a>)</p>

### Executing program

* Plug in your device
* BashBunny will use the following code to find and execute the required Powershell script
```
powershell -NoP -NonI -W h ".((gwmi win32_volume -f 'label=''BashBunny''').Name+'payloads\\$SWITCH_POSITION\rr\rr.ps1')"
```
